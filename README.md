# Loan Analysis & Prediction Project

This project was completed as part of the DecodeLabs Data Science Internship.

The project focuses on analyzing customer loan data, discovering insights, visualizing trends, and building machine learning models to predict loan repayment status.

---
## Dataset

The dataset used in this project is stored externally due to file size limitations.

📥 Download here:
https://drive.google.com/drive/folders/166gl_x1eEBkADa_XH4_AiZc1aXQsjGLe?usp=sharing

# Project Tasks

## Task 1 — Data Understanding
- Understanding dataset structure
- Exploring columns and data types
- Statistical summaries

## Task 2 — Data Cleaning & Preprocessing
- Handling missing values
- Removing duplicates
- Encoding categorical variables
- Preparing data for analysis

## Task 3 — Exploratory Data Analysis (EDA)
- Finding trends and patterns
- Correlation analysis
- Group analysis and pivot tables

## Task 4 — Data Visualization
- Count plots
- Histograms
- Pie charts
- Correlation heatmaps

## Task 5 — Predictive Modeling
Machine learning models used:
- Logistic Regression
- K-Nearest Neighbors (KNN)
- DecisionTreeClassifier
- RandomForestClassifier
- ExtraTreesClassifier
- AdaBoostClassifier
- GradientBoostingClassifier
- XGBClassifier
- CatBoostClassifier
- LGBMClassifier
